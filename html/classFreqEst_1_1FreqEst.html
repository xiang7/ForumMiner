Detailed Description

This is the class to count frequency of patterns from a corpus.

It also computes importance metrics, RIDF, TFIDF, MI. It operates on one set of patterns. Multiple calls of search functions can be called on different strings/corpus. Dictionary persists across these search calls. Call export to file or get frequency to retrieve the results
